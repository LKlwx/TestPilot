📖 使用指引

UI 自动化测试通过“添加步骤”来构建操作流程：
1. 选择动作（点击/输入/等待）→ 选择定位方式（ID/XPath/CSS等）→ 填写定位值或内容 → 点击“添加步骤”
2. 重复上述操作，系统会自动生成操作步骤（每行一个步骤，无需手动换行）
3. 示例：百度搜索 Python
    ① 动作选“输入”，定位选“id”，值填“kw=Python”→ 添加步骤 → 生成 [input] [id] kw=Python
    ② 动作选“点击”，定位选“id”，值填“su”→ 添加步骤 → 生成 [click] [id] su
4. 最后点击“保存用例”即可
注意：步骤文本框里的内容可以手动编辑，但格式必须为 [动作] [定位方式] 值，每行一个步骤。
