# TestPilot 智能自动化测试平台
## 在线演示地址
https://testpilot-production-da60.up.railway.app/
## 项目介绍
TestPilot 是基于 Python Flask 自主开发的**一站式轻量级自动化测试平台**，采用前后端不分离架构，按照路由层、服务层、模型层、工具层进行工程化分层设计。平台实现了**用户权限管理、接口自动化测试、UI 自动化测试、性能压测、AI 智能测试辅助**五大核心能力，支持用例管理、自动化执行、测试报告生成、数据可视化看板等完整基础流程，是面向测试开发岗位的实战型个人项目。

## 项目亮点
- 完善的基础权限体系：支持用户登录、注册、admin/tester 双角色权限控制
- 标准化后端结构：统一响应封装、全局异常处理、代码模块化解耦
- 核心测试能力全覆盖：集成接口、UI、性能三大常用自动化测试模块
- AI 辅助测试：基于规则引擎实现用例自动生成与失败日志智能诊断
- 可视化数据看板：展示用例统计、通过率趋势、性能指标等关键数据
- 全部功能基于本地环境运行，无第三方云服务依赖，轻量易部署

## 技术栈
- 后端：Python 3 + Flask
- 数据库：SQLite
- ORM：Flask-SQLAlchemy
- 身份认证：Flask-JWT-Extended、Flask-Login
- 前端：HTML + CSS + JavaScript + ECharts
- 接口自动化：Requests
- UI 自动化：Selenium（无头模式）
- 性能测试：Python 线程池并发
- AI 模块：本地规则匹配引擎
- 其他：系统操作日志、统一响应封装、全局异常捕获

## 数据模型说明
项目共设计 10 张核心数据表，全部持久化存储：
1. **User**：用户信息、角色、密码
2. **TestCase**：接口测试用例
3. **TestReport**：接口测试报告
4. **UICase**：UI 自动化用例
5. **UIReport**：UI 测试报告
6. **PerformanceCase**：性能测试用例
7. **PerformanceReport**：性能测试指标报告
8. **TestTask**：定时任务（表结构已定义，功能待实现）
9. **AIAgentTask**：AI 操作记录（已落库，前端页面待实现）
10. **SysOperationLog**：系统操作日志（已落库，前端页面待实现）

## 功能模块说明
### 1. 用户权限与登录模块
- 实现用户登录、注册、JWT 身份认证
- 支持 admin 和 tester 双角色权限控制
- 已实现完整的用户列表、角色管理及操作日志审计功能
- 关键操作（登录、删除、修改）自动写入系统日志

### 2. 控制台数据看板
- 展示用例总数、接口/UI/性能用例分布
- 统计整体测试通过率
- 展示近 7 天通过率变化趋势图
- 展示最近 3 次性能测试的 QPS、平均响应时间

### 3. 接口自动化测试模块
- 支持用例新增、删除、列表查询
- 支持配置请求方法、请求头、JSON 请求体、预期关键字
- 支持单条用例执行与批量执行
- 自动生成测试报告，支持报告列表与详情查看
- **✨ 进阶能力**：
  - 接口链路测试：创新性地实现了基于内存变量池的上下文传递机制，支持通过 JSONPath 提取响应数据并动态注入后续请求，解决了登录 Token 传递等经典痛点。
  - 高并发回归：引入 ThreadPoolExecutor 线程池模型，将批量用例执行模式从串行升级为并行，在保证 SQLite 数据一致性的前提下，显著缩短了大规模回归测试的耗时。
  - 工程化规范：遵循 RESTful 风格设计 API，实现了统一的全局异常捕获与标准化响应封装，提升了前后端交互的稳定性。
### 4. UI 自动化测试模块
- 支持 UI 用例新增、删除、列表查询
- 支持配置测试 URL、操作步骤、元素定位信息
- 基于 Selenium 无头浏览器执行自动化操作
- 自动解析并执行输入、点击等文本步骤
- 自动生成并保存 UI 测试报告，支持列表与详情查看

### 5. 性能测试模块
- 支持性能用例新增、删除、列表查询
- 支持自定义并发数、总请求数、请求配置
- 基于线程池实现多线程并发压测
- 自动统计 QPS、响应时间、成功率等指标
- 生成性能测试报告并支持查看

### 6. AI 智能测试模块
- 根据业务场景自动生成接口测试用例
- 根据业务场景自动生成 UI 测试用例
- 自动分析测试失败日志并给出原因与解决方案
- 支持将 AI 生成的用例一键保存至用例库
- AI 操作记录已保存至数据库（**前端历史查看页面待实现**）
